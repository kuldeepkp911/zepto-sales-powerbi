# 🛒 Zepto Sales Analysis Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive **Power BI dashboard** analyzing sales data from a quick-commerce platform inspired by Zepto (10-minute grocery delivery model).

The dashboard provides insights into product performance, category trends, pricing, and stock availability to support data-driven decision-making.

---

## 🎯 Objectives

* Analyze overall sales and revenue performance
* Identify top-selling products and categories
* Monitor stock availability and out-of-stock items
* Understand pricing and discount trends
* Build an interactive dashboard for business insights

---

## 🛠️ Tools & Technologies

* Power BI
* Microsoft Excel
* Data Visualization
* Data Cleaning & Transformation

---

## 📂 Dataset

* **zepto_dataset.xlsx** – Contains product details, categories, pricing, stock, and sales data

## 📊 Dashboard Features

### 🔹 Key Metrics (KPIs)

* 💰 Total Revenue: ₹12bn
* 📦 Total Products Sold: 796K
* 💵 Average Selling Price: 14.19K
* ⚠️ Out of Stock Items: 453

#### 📌 Stock Availability

#### 📌 Top Products Availability

## 🚀 How to Use

1. Download the `.pbix` file
2. Open in **Power BI Desktop**
3. Explore dashboards using filters:

   * Product Name
   * Category
4. Interact with charts for deeper insights

## 💡 Key Insights

* A few categories dominate total sales volume
* Certain products consistently outperform others
* Stock management plays a crucial role in sales performance
* Discount strategies impact category-level sales
